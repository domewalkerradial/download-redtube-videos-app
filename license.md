## 🔍 Troubleshooting Matrix

* **❌ Extension fails to see the stream?** Make sure you have clicked play. The tool must wait for the platform media engine to initialize the stream pathway.
* **📉 Resolution settings missing?** If a video page only provides a single stream source configuration, the interface can only show one option.
* **🔄 Popup tracking an incorrect video block?** Reload the browser tab, hit play to reset the media pipeline, and reopen the extension menu window.
* **🛑 Downloader stopped halfway?** This is usually due to a transient internet dropout. Check your local Wi-Fi or Ethernet connection and rerun the process.
* **🔐 Account locks or paywalled content?** The tool cannot bypass access tiers. The item must be fully playable in your standard browser session for extraction to succeed.
